# Ryujinx-Nextendo

**The Nextendo Network emulator** — a [Ryujinx](https://github.com/Ryubing/Ryujinx) build with the Nextendo account system and online play built right in.

## Download

Grab the latest build from **[Releases](https://github.com/NextendoNetwork/Ryujinx-Nextendo/releases/latest)**, unzip anywhere, and run `Ryujinx.exe`.

> **Windows · x64 · self-contained** — no .NET install required.

## First launch

A quick-start wizard walks you through:
1. Your **games** folder and **updates/DLC** folder.
2. Your **`prod.keys`** (bring your own).
3. Your **Nextendo account** — required for online play. Create one for free (or log in) at **[nextendo.network](https://nextendo.network)**, then link it. No Nintendo account needed.

Then launch a supported game and look for the online modes — they connect straight to the Nextendo servers.

## Supported games

- 🏎️ **Mario Kart 8 Deluxe** — matchmaking, rooms, races
- 🦑 **Splatoon 2** — Turf War, Salmon Run, private battles, Splatfests
- 🥊 **Super Smash Bros. Ultimate** — online matches

## Updates are mandatory

Online play evolves with the servers, so Ryujinx-Nextendo keeps itself current: when a new version is required, the emulator asks you to update before you can keep using it. Always grab the newest release here.

## Account

A **Nextendo account is required** to play online. Create one for free at **[nextendo.network](https://nextendo.network)** — you get a permanent friend code, a friends list and cross-device presence (emulator ↔ Switch).
